# RSI 相對強弱指標 (Relative Strength Index)

> 由 J. Welles Wilder 於 1978 年發明。
> 衡量價格「內在強度」的動量指標，判斷超買/超賣狀態。

---

## 計算方式

```
RSI = 100 - (100 / (1 + RS))

RS = 平均上漲幅度 / 平均下跌幅度
   = (N 日內上漲日的平均漲幅) / (N 日內下跌日的平均跌幅)
```

### 標準參數
- 週期：**14 日**（Wilder 原始建議）
- RSI 值範圍：0 ~ 100

### 白話文
- RSI 越高 → 近期漲多跌少 → 價格「強勢」
- RSI 越低 → 近期跌多漲少 → 價格「弱勢」

---

## 超買/超賣判定

### 標準閾值：70/30

| RSI 範圍 | 狀態 | 意義 |
|:---:|:---:|------|
| > 70 | **超買** | 近期漲幅過大，可能即將回調 |
| 30 ~ 70 | **中性** | 正常波動範圍 |
| < 30 | **超賣** | 近期跌幅過大，可能即將反彈 |

### 保守閾值：80/20
- 適合強趨勢市場
- 減少假訊號，但可能錯過部分機會

### 根據市場調整

| 市場狀態 | 超買閾值 | 超賣閾值 |
|----------|:---:|:---:|
| 多頭市場 | 80 | 40 |
| 空頭市場 | 60 | 20 |
| 盤整市場 | 70 | 30 |

**重要：** 在強勢多頭中，RSI 可以在 70 以上待很久——超買不代表馬上就跌！

---

## RSI 的五種訊號

### 1. 超買/超賣反轉
```
RSI > 70 後回落跌破 70 → 賣出訊號
RSI < 30 後回升突破 30 → 買入訊號

注意：是「回落/回升」後的訊號，不是「剛觸及」就操作！
```

### 2. 中軸交叉 (50 線)
```
RSI 突破 50 → 短期動量轉多
RSI 跌破 50 → 短期動量轉空

用法：作為趨勢方向的輔助確認
```

### 3. RSI 背離 (Divergence)

#### 頂背離（看跌）
```
價格：高    更高
       /\    /\
      /  \  /  \

RSI：高    更低
       /\
      /  \  /\
     /    \/  \

→ 價格新高但動量下降 → 上漲力道衰竭
```

#### 底背離（看漲）
```
價格：低    更低
      \  /  \  /
       \/    \/

RSI：低    更高
      \  /
       \/    /\
            /  \

→ 價格新低但動量回升 → 下跌力道衰竭
```

### 4. Failure Swing（失敗擺動）

比背離更強的反轉訊號。

#### 多頭 Failure Swing（買入）
```
RSI:
1. RSI 跌到 30 以下（A 點）
2. RSI 反彈到某個高點（B 點）
3. RSI 再次下跌但不破 A 點
4. RSI 突破 B 點 → 買入！

         B
        /|\
       / | \     /
      /  |  \   / ← 突破 B 點 = 買入
     /   |   \ /
    A    |    C (不破 A)
```

#### 空頭 Failure Swing（賣出）
```
RSI:
1. RSI 漲到 70 以上（A 點）
2. RSI 回落到某個低點（B 點）
3. RSI 再次上漲但不破 A 點
4. RSI 跌破 B 點 → 賣出！
```

### 5. RSI 趨勢線
- 可以在 RSI 圖形上畫趨勢線
- RSI 突破下降趨勢線 → 動量轉多
- RSI 跌破上升趨勢線 → 動量轉空
- 有時 RSI 的趨勢線比價格的趨勢線更早被突破

---

## 週期參數選擇

| 週期 | 特點 | 適用場景 |
|:---:|------|----------|
| **7** | 非常敏感，多雜訊 | 極短線、當沖 |
| **9** | 較敏感 | 短線交易 |
| **14** | 標準設定（推薦） | 通用 |
| **21** | 較平滑 | 波段操作 |
| **25** | 很平滑，較少訊號 | 中長線 |

---

## 實戰交易規則

### 基本策略
```
買入條件：
  RSI 從低於 30 回升突破 30
  + 成交量開始放大
  + MACD 出現金叉確認

賣出條件：
  RSI 從高於 70 回落跌破 70
  + 出現頂背離
  + MACD 出現死叉確認
```

### RSI + 布林通道組合
```
強力買入：RSI < 30 + 價格觸及布林下軌
強力賣出：RSI > 70 + 價格觸及布林上軌
```

---

## 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| RSI 一超過 70 就賣出 | 強趨勢中 RSI 可以長期在 70 以上 |
| RSI 一低於 30 就買入 | 弱勢中 RSI 可以長期在 30 以下 |
| 只看超買超賣 | 背離和 Failure Swing 是更強的訊號 |
| 不考慮大趨勢 | 先確認趨勢方向，再用 RSI 找進場點 |
