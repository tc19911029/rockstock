# 移動平均線 (Moving Averages, MA)

> 最基礎、最廣泛使用的趨勢追蹤工具。
> 核心功能：平滑價格波動，判斷趨勢方向和支撐/壓力位。

---

## SMA vs EMA

### SMA (Simple Moving Average) 簡單移動平均線
- **計算：** 最近 N 日收盤價的算術平均
- **公式：** SMA = (P1 + P2 + ... + Pn) / N
- **特點：** 反應較慢，但更穩定，不容易被單日極端值影響

### EMA (Exponential Moving Average) 指數移動平均線
- **計算：** 給近期價格更高的權重
- **公式：** EMA = 今日收盤 × K + 昨日 EMA × (1-K)，其中 K = 2/(N+1)
- **特點：** 反應較快，能更早捕捉趨勢變化

### 哪個比較好？

| 比較 | SMA | EMA |
|------|-----|-----|
| 反應速度 | 慢 | 快 |
| 雜訊過濾 | 好 | 一般 |
| 適合場景 | 中長線趨勢判斷 | 短線交易、快速反應 |
| 假訊號 | 較少 | 較多 |

**建議：** 短線用 EMA，中長線用 SMA。或者兩者搭配使用。

## 常用週期與含義

| 週期 | 意義 | 常用場景 |
|:---:|------|----------|
| **5 日** | 一週均線（週線） | 極短線交易 |
| **10 日** | 兩週均線 | 短線交易 |
| **20 日** | 月線 | 短中線交易的重要參考 |
| **60 日** | 季線 | 中線趨勢判斷的關鍵 |
| **120 日** | 半年線 | 中長線分水嶺 |
| **240 日** | 年線 | 長線多空分界的「生命線」 |

### 台股常用組合
- **短線交易：** 5MA + 10MA + 20MA
- **波段操作：** 20MA + 60MA
- **長線投資：** 60MA + 120MA + 240MA

## 黃金交叉與死亡交叉

### 黃金交叉 (Golden Cross) — 買入訊號
```
短期均線由下往上穿越長期均線

價格 ──────────╱── 短期 MA
              ╱
長期 MA ────╱────────

→ 多頭趨勢開始，考慮買入
```

**經典範例：** 5MA 上穿 20MA，或 50MA 上穿 200MA

### 死亡交叉 (Death Cross) — 賣出訊號
```
短期均線由上往下穿越長期均線

長期 MA ────╲────────
              ╲
價格 ──────────╲── 短期 MA

→ 空頭趨勢開始，考慮賣出
```

**注意事項：**
- 交叉訊號是**滯後指標**，等到交叉出現時趨勢已經走了一段
- 在盤整行情中容易出現反覆交叉的**假訊號**
- 建議搭配其他指標（量能、MACD）確認

## 多頭排列與空頭排列

### 多頭排列
```
5MA > 10MA > 20MA > 60MA

均線從上到下依次排列：短期在上、長期在下
→ 強勢多頭趨勢，持股不動
```

### 空頭排列
```
5MA < 10MA < 20MA < 60MA

均線從上到下依次排列：長期在上、短期在下
→ 強勢空頭趨勢，不要接刀
```

## MA 作為支撐與壓力

### 上漲趨勢中：MA 是支撐
- 股價回調到重要均線（如 20MA、60MA）往往會獲得支撐反彈
- 支撐強度：MA 越長期，支撐力道越強
- 如果跌破某條重要均線，下一條均線就變成新的支撐

### 下跌趨勢中：MA 是壓力
- 股價反彈到重要均線往往會遇到壓力回落
- 壓力強度：MA 越長期，壓力越大
- 如果突破某條重要均線，下一條均線就變成新的壓力

### 實戰技巧：均線扣抵

均線的方向取決於「新加入的價格」和「被扣除的價格」的關係：
- 如果新加入的價格 > 被扣除的價格 → 均線上彎
- 如果新加入的價格 < 被扣除的價格 → 均線下彎
- 利用這個原理可以提前預判均線方向的轉變

## 葛蘭碧八大法則

經典的均線買賣法則：

### 四個買入訊號
1. **均線從下降轉為水平或上升，價格由下向上突破均線** → 買入
2. **價格跌破上升中的均線後迅速反彈站回** → 買入（回測支撐成功）
3. **價格在均線上方回跌但未跌破均線又上漲** → 加碼買入
4. **價格急跌遠離均線（乖離率過大）** → 短線反彈買入

### 四個賣出訊號
5. **均線從上升轉為水平或下降，價格由上向下跌破均線** → 賣出
6. **價格突破下降中的均線後迅速回落** → 賣出（假突破）
7. **價格在均線下方反彈但未突破均線又下跌** → 加碼賣出
8. **價格急漲遠離均線（乖離率過大）** → 短線獲利了結

## 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 盤整時用均線交叉做交易 | 盤整行情中均線交叉頻繁，應暫停使用 |
| 只看一條均線 | 至少看 2~3 條不同週期的均線 |
| 忽略均線斜率 | 均線角度越陡代表趨勢越強 |
| 生搬硬套黃金/死亡交叉 | 必須搭配量能和其他指標確認 |
