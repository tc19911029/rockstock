## 核心觀念速記

### 指標四大分類（選指標時不要選同類的）

| 分類 | 功能 | 代表指標 |
|------|------|----------|
| **趨勢類** | 判斷方向 | MA、MACD |
| **動量類** | 衡量強弱 | RSI、KD |
| **波動類** | 界定區間 | 布林通道 |
| **成交量類** | 確認真偽 | OBV、成交量 |

### 最強組合公式

```
勝率最高的訊號 = 趨勢確認 + 動量確認 + 量能確認

範例：
  MA 多頭排列（趨勢↑）
  + MACD 金叉（動量↑）
  + 成交量放大（量能確認）
  = 高勝率買入訊號
```

### 新手三大禁忌

1. **只看一個指標** — 任何單一指標都會有假訊號
2. **逆勢操作** — 先判斷大趨勢，順勢交易勝率遠高於逆勢
3. **沒有停損** — 技術分析告訴你「在哪裡你的判斷就錯了」，到了就要停損
