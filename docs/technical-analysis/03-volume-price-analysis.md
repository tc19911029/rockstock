# 量價關係 (Volume-Price Analysis)

> 「價格可以被操縱，但成交量反映了市場參與者的真實意圖。」
> 量價分析是識別法人動向、判斷趨勢真偽最核心的方法。

---

## 四種基本量價組合

| 價格 | 成交量 | 意義 | 後續判斷 |
|:---:|:---:|------|----------|
| **↑ 漲** | **↑ 增** | 多方確認：買盤強勁 | 趨勢健康，可持續 |
| **↑ 漲** | **↓ 縮** | 多方警訊：動能不足 | 上漲可能即將結束 |
| **↓ 跌** | **↑ 增** | 空方確認：賣壓沉重 | 下跌趨勢確認 |
| **↓ 跌** | **↓ 縮** | 空方衰竭：賣壓減輕 | 可能即將築底反轉 |

### 口訣
```
量增價漲 → 多頭健康，安心持有
量縮價漲 → 小心！可能是「無量上漲」的陷阱
量增價跌 → 快跑！大量賣出中
量縮價跌 → 賣壓減緩，注意落底訊號
```

## Wyckoff 市場週期

Richard Wyckoff 將市場分為四個階段，這是理解法人操作的核心框架：

```
價格
 ^
 |          ┌── 派發期 ──┐
 |         /    (Distribution)  \
 |        /                      \
 |       / 拉升期                 \ 下跌期
 |      / (Markup)                \ (Markdown)
 |     /                           \
 |    /                             \
 |   ┌── 吸籌期 ──┐                 ┌── 吸籌期 ──┐
 |   (Accumulation)                  (Accumulation)
 └────────────────────────────────────────────→ 時間
```

### 1. 吸籌期 (Accumulation)
- **價格：** 低位橫盤整理，波動幅度小
- **成交量：** 逐漸增加，但價格不跌
- **意義：** 聰明錢在大眾悲觀時悄悄買入
- **識別方法：** 下跌量縮 + 偶爾放量但價格不再破底

### 2. 拉升期 (Markup)
- **價格：** 突破盤整區間，開始上漲趨勢
- **成交量：** 明顯放大，尤其在突破時
- **意義：** 機構已累積足夠部位，準備推升價格
- **識別方法：** 放量突破壓力位 + 回測不破

### 3. 派發期 (Distribution)
- **價格：** 高位橫盤，似漲非漲
- **成交量：** 量大但價不漲（大量換手）
- **意義：** 聰明錢在散戶狂熱時悄悄賣出
- **識別方法：** 放量但價格上不去 + 高位頻繁震盪

### 4. 下跌期 (Markdown)
- **價格：** 跌破支撐，加速下跌
- **成交量：** 恐慌性放量
- **意義：** 機構已出貨完畢，散戶恐慌賣出
- **識別方法：** 大量跌破關鍵支撐 + 反彈量縮

## 「努力 vs 結果」法則

> 成交量代表「努力」(Effort)，價格變動代表「結果」(Result)。
> 當努力和結果不匹配時，就是重要的警示訊號。

| 努力（量） | 結果（價） | 判讀 |
|:---:|:---:|------|
| 量很大 | 價格漲很多 | 正常 — 努力匹配結果 |
| 量很大 | 價格只漲一點 | **異常** — 有人在高位大量出貨 |
| 量很大 | 價格只跌一點 | **異常** — 有人在低位大量吃貨 |
| 量很小 | 價格大漲 | **異常** — 缺乏支撐的假突破 |

## 如何偵測法人/主力動向

### 吸籌的蛛絲馬跡
1. **長期下跌後，量縮到極低** — 沒人賣了（浮動籌碼被收乾）
2. **低位突然放量但價格不漲** — 大單在買，但故意不拉價格
3. **下跌量縮，反彈量增** — 賣壓減輕，買盤增加
4. **盤整時間越長，後續漲幅越大** — 籌碼收集得越完整

### 派發的蛛絲馬跡
1. **高位放量滯漲** — 大量成交但價格上不去
2. **連續大量陰線** — 機構在大量出貨
3. **利多消息出來反而下跌** — 內行人趁利多出貨
4. **上漲量縮、下跌量增** — 買盤退潮、賣盤湧入

## 常用量能指標

### OBV (On-Balance Volume) 能量潮
- **計算：** 漲日量累加，跌日量累減
- **用法：** OBV 趨勢向上 → 買盤佔優；OBV 趨勢向下 → 賣盤佔優
- **重點：** 價格創新高但 OBV 沒有創新高 → **量價背離**，危險訊號

### A/D Line (Accumulation/Distribution) 蓄積/派發線
- **計算：** 考慮收盤價在當日高低點的相對位置
- **用法：** A/D 線上升 → 吸籌中；A/D 線下降 → 派發中
- **優勢：** 比 OBV 更精確，因為考慮了盤中的多空拉鋸

### VWAP (Volume Weighted Average Price) 成交量加權平均價
- **計算：** 每筆成交金額總和 / 總成交量
- **用法：** 當日交易中，價格在 VWAP 上方 → 多頭佔優
- **重點：** 法人常以 VWAP 作為當日買賣的基準價格

## 突破的量能確認

### 真突破 vs 假突破

| 特徵 | 真突破 | 假突破 |
|------|--------|--------|
| 成交量 | 比 20 日均量多 50% 以上 | 量能平平或萎縮 |
| 價格行為 | 突破後站穩，回測不破 | 突破後迅速回落 |
| RSI/MACD | 動量指標確認 | 動量背離 |
| 多時間框架 | 大小週期都確認 | 只在小週期突破 |

### 三步確認法
1. **看量：** 突破當日成交量是否比均量大 50% 以上
2. **看站穩：** 突破後是否能收在關鍵位之上（至少收盤確認）
3. **看回測：** 回測支撐時是否量縮且不破

## 實戰口訣

```
1. 天量天價 → 頭部訊號，小心
2. 地量地價 → 底部訊號，注意
3. 量先價行 → 成交量的轉變比價格更早
4. 量在價先 → 量能先萎縮，價格才開始跌
5. 放量突破 → 看好；縮量突破 → 小心陷阱
6. 下跌放量 → 繼續跌；下跌縮量 → 接近底部
```

## 優勢與侷限

### 優勢
- 成交量無法造假（相對價格而言）
- 提供價格走勢背後的「真實意圖」
- 可以提前發現趨勢轉變
- 適用於所有市場和時間框架

### 侷限
- 需要一定的經驗來判讀（主觀成分存在）
- 某些市場（如外匯）沒有真實的集中成交量數據
- 極端行情中（跳空、漲停）可能失真
- 需要搭配價格型態一起判讀
