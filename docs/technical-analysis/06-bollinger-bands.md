# 布林通道 (Bollinger Bands)

> 由 John Bollinger 於 1980 年代發明。
> 利用標準差動態界定股價波動區間，是抓反彈與判斷突破的利器。

---

## 組成元素

| 元素 | 計算方式 | 意義 |
|------|----------|------|
| **中軌** | 20 日 SMA | 趨勢的中心線 |
| **上軌** | 中軌 + 2 × 標準差 | 壓力帶 / 超買區 |
| **下軌** | 中軌 - 2 × 標準差 | 支撐帶 / 超賣區 |

### 標準參數
- 週期：20
- 標準差倍數：2
- 統計上，約 95% 的價格會落在上下軌之間

```
上軌 ─────────╲    ╱─────────  壓力
                ╲  ╱
中軌 ────────── ╳ ──────────  趨勢中心
                ╱  ╲
下軌 ─────────╱    ╲─────────  支撐
              收窄    擴張
             (Squeeze)
```

---

## 核心概念：均值回歸

> 布林通道的基本假設：價格傾向於回到平均值附近。

- 觸及上軌 → 可能超漲，注意回調
- 觸及下軌 → 可能超跌，注意反彈
- 價格長期在中軌附近波動是「正常」狀態

**但要注意：** 在強趨勢中，價格可以「沿著軌道行走」（Band Walk），均值回歸暫時失效。

---

## 四大交易形態

### 1. 擠壓 (Bollinger Squeeze)

```
上軌 ──────────────────────
中軌 ──────════════════────   ← 通道收窄
下軌 ──────────────────────
            ↑
        Squeeze（擠壓）
        = 波動率極低
        = 大行情即將來臨
```

- **識別：** 上下軌距離收到近期最窄
- **意義：** 市場在「蓄力」，即將出現大幅波動
- **策略：** 不預測方向，等突破後跟進
- **注意：** 擠壓可以持續很久，不要急

### 2. 帶狀行走 (Band Walk)

```
上軌 ──────╱╱╱╱╱╱────────
中軌 ────╱╱╱╱╱╱╱╱────────   ← 價格沿著上軌走
下軌 ──╱╱────────────────

→ 強勢多頭，不要輕易做空
```

- 多頭 Band Walk：價格持續觸及或超越上軌
- 空頭 Band Walk：價格持續觸及或跌破下軌
- **重要：** Band Walk 期間不要逆勢操作！

### 3. 雙底 W 底 (W-Bottom)

```
       /\
      /  \         /\
     /    \       /  \  ← 第二次觸底在下軌之上
    /      \     /    \     且 RSI 沒有創新低
───/────────\───/──────\───── 下軌
              \/
         第一次觸底
         跌破下軌

→ 底部確認，看漲訊號
```

### 4. 雙頂 M 頭 (M-Top)

```
───\────────/───\──────/───── 上軌
    \      /     \    /
     \    /       \  /  ← 第二次觸頂在上軌之下
      \  /         \/      且 RSI 沒有創新高
       \/
    第一次觸頂
    突破上軌

→ 頭部確認，看跌訊號
```

## %B 指標

%B 告訴你價格在布林通道中的相對位置。

**公式：** %B = (收盤價 - 下軌) / (上軌 - 下軌)

| %B 值 | 位置 | 意義 |
|:---:|------|------|
| > 1.0 | 超過上軌 | 極度超買 |
| = 1.0 | 在上軌 | 超買邊界 |
| = 0.5 | 在中軌 | 中性 |
| = 0.0 | 在下軌 | 超賣邊界 |
| < 0.0 | 低於下軌 | 極度超賣 |

---

## Bandwidth 指標

衡量布林通道的寬度，用來識別擠壓。

**公式：** Bandwidth = (上軌 - 下軌) / 中軌

- **Bandwidth 低 → 擠壓中** → 準備大行情
- **Bandwidth 高 → 波動大** → 趨勢可能接近尾聲
- 歷史最低的 Bandwidth 往往預示最大的行情

---

## 實戰交易規則

### 均值回歸策略（適合盤整行情）
```
買入：價格觸及下軌 + RSI < 30 + 成交量萎縮
賣出：價格觸及上軌 + RSI > 70 + 成交量萎縮
停損：跌破下軌一定幅度（如 1~2%）
```

### 突破策略（適合趨勢行情）
```
買入：擠壓後價格突破上軌 + 成交量放大
賣出：擠壓後價格跌破下軌 + 成交量放大
確認：MACD 方向一致
```

### 關鍵判斷：盤整還是趨勢？
- 布林帶收窄 → 盤整 → 用均值回歸策略
- 布林帶擴張 → 趨勢 → 用突破/順勢策略
- Band Walk 出現 → 強趨勢 → 不要逆勢

## 參數調整

| 場景 | 週期 | 標準差 |
|------|:---:|:---:|
| 標準設定 | 20 | 2 |
| 短線交易 | 10 | 1.5 |
| 長線投資 | 50 | 2.5 |
| 高波動市場 | 20 | 2.5 |

---

## 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 一觸上軌就賣出 | 強趨勢中會沿著上軌走，要先判斷趨勢 |
| 一觸下軌就買入 | 空頭趨勢中會沿著下軌走，要先判斷趨勢 |
| 忽略擠壓訊號 | 擠壓是布林通道最重要的形態之一 |
| 不搭配其他指標 | 布林通道 + RSI + MACD 效果最佳 |
