# 波浪理論 (Elliott Wave Theory)

> 由 Ralph Nelson Elliott 於 1930 年代提出。
> 核心觀點：市場走勢反映群體心理，呈現可預測的「波浪」模式。

---

## 基本結構：5 + 3

市場以 **5 浪推動 + 3 浪修正** 的模式反覆循環。

```
5 浪推動（順勢）          3 浪修正（逆勢）

        5
       / \                 A
      /   \               / \
    3/     \             /   \
   / \      \           /     B
  /   \      \         /     / \
 /     4      \       /     /   \
1      |       \     /     /     C
|      |        \   /     /
|      |         \ /     /
0      2
```

### 推動浪（Impulse Wave）1-2-3-4-5

| 浪 | 特徵 | 心理狀態 |
|:---:|------|----------|
| **第 1 浪** | 趨勢開始，多數人還不相信 | 聰明錢開始佈局 |
| **第 2 浪** | 回調（但不會跌破第 1 浪起點） | 懷疑者認為「又要跌了」 |
| **第 3 浪** | 最強最長的一浪，量能最大 | 趨勢確認，大眾湧入 |
| **第 4 浪** | 獲利了結回調 | 部分人開始獲利出場 |
| **第 5 浪** | 最後的衝刺，動能減弱 | 狂熱階段，散戶追高 |

### 修正浪（Corrective Wave）A-B-C

| 浪 | 特徵 | 心理狀態 |
|:---:|------|----------|
| **A 浪** | 下跌開始，但多數人認為是回調 | 「逢低買入」心態 |
| **B 浪** | 反彈，但力道不足 | 「果然只是回調」（陷阱） |
| **C 浪** | 真正的下殺，跌幅往往最大 | 恐慌性賣出 |

---

## 三條不可違反的規則

這三條是「鐵律」，違反任何一條就代表數浪數錯了：

### 規則 1：第 2 浪不能回調超過第 1 浪的 100%
```
✅ 正確：第 2 浪在第 1 浪起點之上結束
❌ 錯誤：第 2 浪跌破了第 1 浪的起點 → 重新數浪
```

### 規則 2：第 3 浪不能是最短的推動浪
```
✅ 正確：第 3 浪比第 1 浪或第 5 浪長（通常是最長的）
❌ 錯誤：第 3 浪比第 1 浪和第 5 浪都短 → 重新數浪
```

### 規則 3：第 4 浪不能進入第 1 浪的價格區間
```
✅ 正確：第 4 浪的最低點高於第 1 浪的最高點
❌ 錯誤：第 4 浪跌破了第 1 浪的頂部 → 重新數浪
（例外：斜角三角形態中可以重疊）
```

---

## 重要指導原則（非鐵律，但很實用）

### 交替原則 (Alternation)
- 如果第 2 浪是**急跌**（簡單修正），第 4 浪通常是**橫盤**（複雜修正）
- 反之亦然

### 等長原則 (Equality)
- 第 1 浪、第 3 浪、第 5 浪中，通常有兩浪等長
- 最常見：第 1 浪 = 第 5 浪

### 通道原則 (Channeling)
- 連接第 2 浪和第 4 浪的低點畫一條線
- 平行線經過第 3 浪的頂部
- 第 5 浪通常在通道上緣附近結束

## 費波那契比率與各浪目標

費波那契數列（1, 1, 2, 3, 5, 8, 13, 21...）的比率是波浪理論的數學基礎。

### 常用費波那契比率
`0.236, 0.382, 0.500, 0.618, 0.764, 1.000, 1.618, 2.618`

### 各浪的典型費波那契關係

| 浪 | 回調/延伸 | 常見比率 |
|:---:|-----------|----------|
| **第 2 浪** | 回調第 1 浪 | **61.8%**、50%、76.4% |
| **第 3 浪** | 延伸第 1 浪 | **161.8%**、200%、261.8% |
| **第 4 浪** | 回調第 3 浪 | **38.2%**、50% |
| **第 5 浪** | 相對第 1 浪 | **100%**（等長）、61.8% |
| **A 浪** | 回調第 5 浪 | 38.2%、50%、61.8% |
| **B 浪** | 回調 A 浪 | 50%、61.8%、76.4% |
| **C 浪** | 相對 A 浪 | **100%**、161.8% |

### 如何使用

```
已知第 1 浪從 100 漲到 120（漲幅 20 元）

預測第 3 浪目標：
  第 2 浪低點 + 第 1 浪漲幅 × 1.618
  = 第 2 浪低點 + 20 × 1.618
  = 第 2 浪低點 + 32.36

預測第 2 浪支撐：
  120 - 20 × 0.618 = 120 - 12.36 = 107.64
```

## 常見修正浪形態

### 鋸齒形 (Zigzag) — A-B-C
- 最常見的修正形態
- A 浪和 C 浪是推動浪，B 浪是修正浪
- 修正幅度較深，通常回調 50%~78.6%

### 平坦形 (Flat) — A-B-C
- B 浪反彈到接近 A 浪起點
- C 浪跌至接近 A 浪終點
- 修正幅度較淺，橫向整理為主

### 三角形 (Triangle) — A-B-C-D-E
- 五個子浪組成收斂三角形
- 通常出現在第 4 浪或 B 浪位置
- 突破方向通常延續原有趨勢

## 實戰建議

### 最佳進場點
1. **在第 2 浪結束時買入** — 搭上最強的第 3 浪
2. **在第 4 浪結束時買入** — 搭上第 5 浪
3. **在 C 浪結束時買入** — 搭上新一輪的第 1 浪

### 停損設定
- 買在第 2 浪低點 → 停損設在第 1 浪起點下方（規則 1 邊界）
- 買在第 4 浪低點 → 停損設在第 1 浪頂部下方（規則 3 邊界）

### 常見錯誤

| 錯誤 | 解決方法 |
|------|----------|
| 強迫數浪去符合自己的看法 | 永遠準備「備選方案」，數浪不是只有一種可能 |
| 混淆「規則」和「指導原則」 | 規則不可違反，指導原則是參考 |
| 只在一個時間框架數浪 | 至少看兩個時間框架，確認浪的層級一致 |
| 忽略比例對稱 | 形狀畸形的浪要懷疑數錯了 |

## 優勢與侷限

### 優勢
- 提供完整的市場週期預測框架
- 費波那契比率給出具體的價格目標
- 可以在任何時間框架使用
- 幫助理解市場心理週期

### 侷限
- 數浪具有主觀性，不同人可能數出不同結果
- 需要大量經驗才能熟練
- 即時數浪困難，往往事後才能確認
- 不適合作為唯一的交易依據
