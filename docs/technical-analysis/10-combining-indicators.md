# 多指標組合策略

> 「最強的技術分析不是只看一個指標，而是多維度的確認。」
> 單一指標勝率約 50~60%，多指標組合可以提升到 65~75%。

---

## 指標分類：避免冗餘

使用多個指標時，**最重要的原則是不要選同類型的指標**。

| 分類 | 功能 | 代表指標 |
|------|------|----------|
| **趨勢類** | 判斷方向 | MA、MACD |
| **動量類** | 衡量強弱 | RSI、KD |
| **波動類** | 界定區間 | 布林通道 |
| **成交量類** | 確認真偽 | OBV、成交量 |

### 冗餘範例（不要這樣做）
```
❌ RSI + KD + Stochastic RSI
   → 三個都是動量指標，本質相同，只會讓你「過度自信」

❌ MA + EMA + WMA
   → 三個都是均線，看到的是同一件事

✅ MACD（趨勢）+ RSI（動量）+ 布林通道（波動）+ 成交量
   → 四個不同維度，真正的多角度確認
```

---

## 指標共振 (Indicator Resonance)

> 當多個不同類型的指標同時給出相同方向的訊號，這就是「共振」。
> 共振出現時，訊號的可靠度大幅提升。

### 經典共振範例：三重確認買入

```
條件同時滿足：
┌──────────────────────────────────────┐
│ 1. MACD 金叉（趨勢轉多）             │ → 趨勢確認 ✓
│ 2. RSI < 30 後回升突破 30（動量轉強） │ → 動量確認 ✓
│ 3. 價格觸及布林下軌反彈（超賣反彈）   │ → 波動確認 ✓
│ 4. 成交量放大（資金進場）             │ → 量能確認 ✓
└──────────────────────────────────────┘
            ↓
    高勝率買入訊號！
```

### 經典共振範例：三重確認賣出

```
條件同時滿足：
┌──────────────────────────────────────┐
│ 1. MACD 死叉或頂背離（趨勢轉弱）     │ → 趨勢警告 ✓
│ 2. RSI > 70 後回落跌破 70（動量減弱） │ → 動量警告 ✓
│ 3. 價格觸及布林上軌回落（超買回調）   │ → 波動警告 ✓
│ 4. 成交量放大下跌（資金出場）         │ → 量能警告 ✓
└──────────────────────────────────────┘
            ↓
    高勝率賣出訊號！
```

### 研究數據
- 使用 MACD + RSI 雙重確認，比單用 MACD 減少約 65% 的假訊號
- 三重確認（趨勢 + 動量 + 波動）比單指標提升約 23% 的準確率

## 趨勢先行原則

> **永遠先判斷大趨勢，再用其他指標找進場點。**

### 三步驟操作框架

```
Step 1: 判斷大趨勢（用日線/週線）
  ┌─────────────────────────────────────┐
  │ 工具：MA 多空排列 / MACD 零軸位置   │
  │ 問題：現在是多頭、空頭、還是盤整？   │
  └─────────────────────────────────────┘
              ↓
Step 2: 等待回調機會（用日線）
  ┌─────────────────────────────────────┐
  │ 多頭趨勢中 → 等回調到支撐位         │
  │ 空頭趨勢中 → 等反彈到壓力位         │
  │ 盤整中 → 等突破                      │
  └─────────────────────────────────────┘
              ↓
Step 3: 精確進場（用小時線/日線指標）
  ┌─────────────────────────────────────┐
  │ 工具：RSI/KD 超賣區金叉              │
  │      K 線反轉型態                    │
  │      布林通道觸及下軌                 │
  └─────────────────────────────────────┘
```

### 為什麼順勢勝率更高？
- 統計上，順勢交易勝率約 55~65%，逆勢交易勝率約 35~45%
- 趨勢行情佔市場時間的 30%，但貢獻了 70% 的獲利機會
- 「不要和趨勢作對」是所有成功交易者的共識

## 量價確認與識別假突破

### 真突破 vs 假突破的判別表

| 判別指標 | 真突破 | 假突破 |
|----------|--------|--------|
| **成交量** | 超過 20 日均量的 150% | 正常或偏低 |
| **收盤位置** | 收在突破位之上 | 無法收在突破位之上 |
| **隔日表現** | 繼續上漲或高位橫盤 | 迅速回落 |
| **RSI/MACD** | 動量指標確認 | 動量指標背離 |
| **多時間框架** | 大小週期都突破 | 只在小週期突破 |

### 識別假突破的三步確認法
```
1. 看量 → 突破日成交量是否大增？（≥ 均量 150%）
2. 看站穩 → 收盤能否站在突破位之上？
3. 看回測 → 如果回測原突破位，是否量縮且不破？

三個都通過 → 真突破，可以進場
任何一個沒通過 → 暫時觀望
```

## 結合基本面與籌碼面

### 技術面 + 基本面 + 籌碼面 = 最穩健的決策

```
┌─────────────────────────────────────────────────┐
│                投資決策三角                        │
│                                                   │
│           基本面（What to buy）                    │
│              公司賺不賺錢？                        │
│              估值便不便宜？                        │
│                  /\                               │
│                 /  \                              │
│                /    \                             │
│    技術面 ────/──────\──── 籌碼面                 │
│   (When)     ────────     (Who)                  │
│   什麼時候買?             誰在買?                  │
│   趨勢/指標               法人/主力動向            │
│                                                   │
│   三面共振 = 最高勝率的投資決策                     │
└─────────────────────────────────────────────────┘
```

### 基本面關注重點
| 指標 | 意義 | 基本標準 |
|------|------|----------|
| EPS 成長 | 公司獲利能力 | 持續成長 |
| P/E 本益比 | 估值水準 | 不要太離譜 |
| 營收成長 | 業績動能 | 正成長 |
| ROE | 股東報酬率 | > 15% 較佳 |

### 籌碼面關注重點
| 指標 | 意義 | 看多訊號 |
|------|------|----------|
| 外資買超 | 國際資金動向 | 連續買超 |
| 投信買超 | 法人研究的結論 | 連續買超 |
| 融資變化 | 散戶槓桿情緒 | 融資減少（洗清浮額） |
| 大戶持股 | 主力態度 | 持股增加 |

### 整合範例
```
理想買入情境：
✅ 基本面：營收連續 3 月成長，EPS 優於預期
✅ 技術面：MACD 金叉 + 突破 60 日均線 + 放量
✅ 籌碼面：外資連續買超 + 投信加碼

理想賣出/避開情境：
⚠️ 基本面：營收衰退，EPS 不如預期
⚠️ 技術面：MACD 頂背離 + 跌破 60 日均線
⚠️ 籌碼面：外資連續賣超 + 融資大增
```

## 新手常犯的七大錯誤

### 1. 只看一個指標就交易
- **問題：** 任何單一指標都有 40~50% 的假訊號率
- **解法：** 至少用 2~3 個不同類型的指標交叉確認

### 2. 使用太多同類指標
- **問題：** RSI + KD + Stochastic 都是動量指標，等於只看一個維度
- **解法：** 趨勢 + 動量 + 波動/量能，各選一個

### 3. 忽略大趨勢
- **問題：** 在空頭市場中一直找買入訊號
- **解法：** 先用 MA/MACD 判斷大趨勢，順勢操作

### 4. 過度最佳化（曲線擬合）
- **問題：** 不斷調整參數讓歷史回測完美，但實戰效果差
- **解法：** 使用標準參數，接受 60~70% 的勝率就已經很好

### 5. 情緒交易
- **問題：** 恐懼時不敢買（超賣），貪婪時不願賣（超買）
- **解法：** 事先寫好交易計劃，機械式執行

### 6. 沒有停損
- **問題：** 虧損時抱著「再等等就會漲」的心態
- **解法：** 技術分析明確告訴你「哪裡判斷就錯了」——到了就停損

### 7. 追漲殺跌
- **問題：** 看到漲了才追進，看到跌了才賣出
- **解法：** 在回調時買入（順勢），在衝高時賣出（不追高）

## 三個實戰交易系統

### 系統 1：趨勢跟隨系統（適合波段）
```
指標組合：20MA + 60MA + MACD + 成交量

買入條件：
  1. 20MA > 60MA（多頭排列）
  2. MACD 在零軸上方金叉
  3. 成交量放大

賣出條件：
  1. MACD 出現頂背離
  2. 或 20MA 跌破 60MA

停損：跌破 60MA
```

### 系統 2：超買超賣系統（適合盤整）
```
指標組合：布林通道 + RSI + 成交量

買入條件：
  1. 價格觸及布林下軌
  2. RSI < 30
  3. 出現 K 線反轉型態（如錘子線）

賣出條件：
  1. 價格觸及布林上軌
  2. RSI > 70
  3. 出現 K 線反轉型態（如射擊之星）

停損：跌破布林下軌 2%
注意：只在盤整行情使用！
```

### 系統 3：多維確認系統（綜合）
```
指標組合：MA + MACD + RSI + 布林通道 + 成交量

買入條件（至少 4 個滿足）：
  □ MA 多頭排列或價格在 60MA 之上
  □ MACD 金叉或柱狀圖翻正
  □ RSI > 50 且非超買
  □ 價格在布林中軌附近或下軌反彈
  □ 成交量放大

賣出條件（至少 3 個滿足）：
  □ MACD 頂背離或死叉
  □ RSI > 70 後回落
  □ 價格觸及布林上軌回落
  □ 成交量異常放大但價格不漲

停損：最近一次擺動低點的下方
```

## 交易紀律清單

每次交易前，問自己這些問題：

```
✅ 大趨勢方向是什麼？（我在順勢嗎？）
✅ 有多少個指標給出同方向訊號？（至少 2 個確認）
✅ 成交量支持嗎？（量能確認）
✅ 我的進場價位是什麼？
✅ 我的停損價位是什麼？（必須設定！）
✅ 我的停利目標是什麼？
✅ 風險報酬比是否 ≥ 1:2？
```
