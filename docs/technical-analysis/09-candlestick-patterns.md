# K 線戰法 (Candlestick Patterns)

> K 線圖起源於 18 世紀日本米市場。
> 直接觀察價格走勢的圖形組合，用於抓極短線的轉折點。

---

## K 線基礎

### 單根 K 線的組成

```
    │  ← 上影線（最高價到收盤/開盤價）
    │
  ┌─┴─┐
  │   │ ← 實體（開盤價到收盤價）
  │   │    紅色/白色 = 收盤 > 開盤（漲）
  └─┬─┘    綠色/黑色 = 收盤 < 開盤（跌）
    │
    │  ← 下影線（開盤/收盤價到最低價）
```

### 影線的意義
- **長上影線** → 上方有賣壓，漲不上去
- **長下影線** → 下方有支撐，跌不下去
- **無影線** → 多方或空方完全主導

## 重要反轉型態

### 單根 K 線反轉

#### 十字線 (Doji)
```
    │
  ──┼──    開盤 ≈ 收盤，實體極小
    │

意義：多空拉鋸，猶豫不決
位置：高位十字線 → 看跌轉折
      低位十字線 → 看漲轉折
```

#### 錘子線 (Hammer)
```
  ┌─┐
  │ │      短實體 + 長下影線（下影線 ≥ 實體 2 倍）
  └─┘      無或極短上影線
    │
    │
    │

出現在下跌趨勢底部 → 看漲反轉
意義：空方試圖殺低但被多方拉回
```

#### 上吊線 (Hanging Man)
```
  ┌─┐
  │ │      形狀和錘子線完全一樣
  └─┘      但出現在上漲趨勢頂部
    │
    │
    │

出現在上漲趨勢頂部 → 看跌反轉
意義：多方力竭，空方開始出手
```

#### 射擊之星 (Shooting Star)
```
    │
    │
    │
  ┌─┐
  │ │      短實體 + 長上影線
  └─┘      無或極短下影線

出現在上漲趨勢頂部 → 看跌反轉
意義：多方衝高但被空方打回
```

### 雙根 K 線反轉

#### 多頭吞噬 (Bullish Engulfing)
```
    ┌─┐
    │●│   第一天：小陰線
    └─┘
  ┌─────┐
  │     │ 第二天：大陽線完全包覆第一天
  │  ○  │
  └─────┘

出現在下跌趨勢底部 → 強力看漲
```

#### 空頭吞噬 (Bearish Engulfing)
```
  ┌─────┐
  │  ○  │ 第一天：小陽線
  └─────┘
    ┌─────┐
    │     │ 第二天：大陰線完全包覆第一天
    │  ●  │
    └─────┘

出現在上漲趨勢頂部 → 強力看跌
```

#### 烏雲蓋頂 (Dark Cloud Cover)
```
  ┌───┐
  │ ○ │   第一天：大陽線
  └───┘
    ┌───┐
    │   │ 第二天：開高後大跌
    │ ● │ 收盤低於第一天實體 50% 以下
    └───┘

出現在上漲趨勢頂部 → 看跌
```

#### 曙光初現 (Piercing Pattern)
```
    ┌───┐
    │ ● │ 第一天：大陰線
    └───┘
  ┌───┐
  │   │   第二天：開低後大漲
  │ ○ │   收盤高於第一天實體 50% 以上
  └───┘

出現在下跌趨勢底部 → 看漲
```

### 三根 K 線反轉

#### 晨星 (Morning Star) — 最經典的底部反轉
```
  ┌───┐
  │ ● │       ┌───┐
  │   │  ┌┐   │ ○ │
  └───┘  └┘   │   │
        小實體  └───┘
  第一天  第二天  第三天
  大陰線  十字/小K 大陽線

三天組合 → 下跌趨勢結束，反轉看漲
第三天的收盤應高於第一天實體的 50% 以上
```

#### 暮星 (Evening Star) — 最經典的頂部反轉
```
        小實體
  ┌───┐  ┌┐   ┌───┐
  │ ○ │  └┘   │   │
  │   │       │ ● │
  └───┘       └───┘
  第一天  第二天  第三天
  大陽線  十字/小K 大陰線

三天組合 → 上漲趨勢結束，反轉看跌
```

#### 三白兵 (Three White Soldiers) — 強力看漲
```
         ┌──┐
      ┌──┤  │
  ┌──┤  ││  │
  │  ││  │└──┘
  │  │└──┘
  └──┘

三根連續大陽線，每根收盤都高於前一根
→ 多頭強勢確認
注意：如果出現在已經大漲之後，可能是最後的衝刺
```

#### 三黑鴉 (Three Black Crows) — 強力看跌
```
  ┌──┐
  │  ├──┐
  └──│  ├──┐
     │  ││  │
     └──┤│  │
        └──┘

三根連續大陰線，每根收盤都低於前一根
→ 空頭強勢確認
```

## 持續型態

### 上升三法 (Rising Three Methods)
```
  ┌──┐              ┌──┐
  │  │  ┌┐ ┌┐ ┌┐   │  │
  │○ │  │●││●││●│   │○ │
  └──┘  └┘ └┘ └┘   └──┘

大陽線 + 三根小陰線（在第一根範圍內）+ 大陽線
→ 多頭趨勢持續
```

### 下降三法 (Falling Three Methods)
```
  ┌──┐  ┌┐ ┌┐ ┌┐   ┌──┐
  │● │  │○││○││○│   │  │
  │  │  └┘ └┘ └┘   │● │
  └──┘              └──┘

大陰線 + 三根小陽線（在第一根範圍內）+ 大陰線
→ 空頭趨勢持續
```

## 量能確認要點

K 線型態必須搭配成交量才有說服力：

| K 線型態 | 理想的量能配合 |
|----------|----------------|
| 多頭吞噬 | 第二天成交量明顯放大 |
| 錘子線 | 下影線期間量能放大（有人在接） |
| 晨星 | 第三天（大陽線）量能最大 |
| 三白兵 | 成交量逐日遞增 |
| 突破型態 | 突破日量能至少比均量大 50% |

**口訣：** 「有量的型態才可信，無量的型態要懷疑。」

## 實戰使用建議

### 高可靠度型態排名
1. **晨星/暮星** — 反轉最經典
2. **多頭/空頭吞噬** — 簡單有效
3. **三白兵/三黑鴉** — 趨勢確認
4. **錘子線/射擊之星** — 常見但需確認

### 使用原則
```
1. K 線型態要放在「對的位置」
   - 底部反轉型態要出現在下跌趨勢的底部
   - 頂部反轉型態要出現在上漲趨勢的頂部
   - 在「盤整中間」出現的反轉型態可靠度低

2. 必須搭配量能確認
   - 有量的型態 > 無量的型態

3. 等待確認
   - 看到型態後，等下一根 K 線確認方向再行動
   - 不要在型態「形成中」就急著進場

4. 搭配支撐壓力
   - 在重要支撐位出現底部反轉 → 更可靠
   - 在重要壓力位出現頂部反轉 → 更可靠
```

---

## 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 只看 K 線型態就交易 | 必須結合趨勢、量能、支撐壓力 |
| 在盤整中找反轉型態 | 反轉型態在趨勢末端才有意義 |
| 忽略時間框架 | 日線型態比分鐘線更可靠 |
| 記太多型態 | 精通 5~6 個高可靠度型態就夠了 |
