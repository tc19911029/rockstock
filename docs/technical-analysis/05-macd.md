# MACD 指標 (Moving Average Convergence Divergence)

> 被稱為「指標之王」，結合了趨勢追蹤與動量分析。
> 由 Gerald Appel 於 1979 年發明。

---

## 組成元素

### 三個組成部分

| 元素 | 計算方式 | 意義 |
|------|----------|------|
| **DIF (快線)** | 12 日 EMA - 26 日 EMA | 短期與長期趨勢的差距 |
| **DEA / Signal (慢線)** | DIF 的 9 日 EMA | DIF 的平滑線 |
| **柱狀圖 (Histogram)** | DIF - DEA | 快慢線的差距，動量的加速/減速 |

### 標準參數
- 快線 EMA：12
- 慢線 EMA：26
- 信號線 EMA：9
- 記作 MACD(12, 26, 9)

### 圖形解讀
```
        DIF (快線)
         /\      /\
        /  \    /  \
───────/────\──/────\───── 零軸
      /      \/      \
     /    DEA (慢線)   \
    /                   \

    ████            ████      ← 柱狀圖（DIF - DEA）
    ████            ████
████████████    ████████████
```

---

## 零軸：多空分界線

| DIF 位置 | 意義 |
|:---:|------|
| DIF > 0（零軸上方） | 短期趨勢強於長期 → **多頭市場** |
| DIF < 0（零軸下方） | 短期趨勢弱於長期 → **空頭市場** |
| DIF 穿越零軸向上 | 趨勢由空轉多 |
| DIF 穿越零軸向下 | 趨勢由多轉空 |

**重要：** 零軸是最重要的參考線。MACD 在零軸上方的金叉比在零軸下方的金叉更可靠。

---

## 金叉與死叉

### 金叉 (Golden Cross) — 買入訊號
```
DIF 由下往上穿越 DEA

觸發條件：DIF 上穿 DEA
訊號強度：
  ★★★ 零軸上方的金叉 → 強勢，多頭確認
  ★★  零軸附近的金叉 → 中等，趨勢轉折
  ★   零軸下方的金叉 → 弱勢，僅是空頭中的反彈
```

### 死叉 (Death Cross) — 賣出訊號
```
DIF 由上往下穿越 DEA

觸發條件：DIF 下穿 DEA
訊號強度：
  ★★★ 零軸下方的死叉 → 強勢，空頭確認
  ★★  零軸附近的死叉 → 中等，趨勢轉折
  ★   零軸上方的死叉 → 弱勢，僅是多頭中的回調
```

---

## 柱狀圖的訊號

柱狀圖 = DIF - DEA，反映的是動量的**加速或減速**。

| 柱狀圖變化 | 意義 |
|------------|------|
| 紅柱越來越長 | 多頭動能加速 → 繼續持有 |
| 紅柱開始縮短 | 多頭動能減弱 → 準備獲利 |
| 由紅轉綠 | 動量轉空 → 賣出訊號 |
| 綠柱越來越長 | 空頭動能加速 → 不要接刀 |
| 綠柱開始縮短 | 空頭動能減弱 → 注意落底 |
| 由綠轉紅 | 動量轉多 → 買入訊號 |

**技巧：** 柱狀圖的變化比金叉/死叉更早出現，可以作為提前預警。

---

## 背離 (Divergence) — 最強訊號

背離是 MACD 最重要的訊號，代表趨勢即將反轉。

### 頂背離（看跌）
```
價格：創新高    再創新高
        /\        /\
       /  \      /  \
      /    \    /    \

MACD：創高      但更低
        /\
       /  \    /\
      /    \  /  \

→ 價格創新高但 MACD 沒有 → 上漲動能衰竭 → 準備下跌
```

### 底背離（看漲）
```
價格：創新低    再創新低
      \    /  \    /
       \  /    \  /
        \/      \/

MACD：創低      但更高
      \    /
       \  /    /\
        \/    /  \

→ 價格創新低但 MACD 沒有 → 下跌動能衰竭 → 準備上漲
```

### 背離使用注意
- 背離出現後不要立刻進場，等待價格確認（如突破趨勢線）
- 連續兩次背離比一次背離更強
- 週線級別的背離比日線級別更可靠
- 背離只是「警告」，不是「保證」

## MACD 實戰系統

### 基本交易規則
```
買入條件（滿足越多越好）：
1. DIF 在零軸上方（多頭市場）
2. DIF 上穿 DEA（金叉）
3. 柱狀圖由負轉正
4. 成交量配合放大

賣出條件（滿足越多越好）：
1. DIF 在零軸下方（空頭市場）
2. DIF 下穿 DEA（死叉）
3. 柱狀圖由正轉負
4. 出現頂背離
```

### 進階技巧：MACD 二次金叉
- 第一次金叉後，出現短暫死叉（回調）
- 隨後再次金叉（且不跌破零軸）
- 第二次金叉的可靠度遠高於第一次
- 代表回調洗盤完畢，主升浪即將開始

## 不同時間框架的參數調整

| 時間框架 | 建議參數 | 適用場景 |
|----------|----------|----------|
| 日線 | (12, 26, 9) 標準 | 中線波段操作 |
| 週線 | (12, 26, 9) | 長線趨勢判斷 |
| 60分鐘 | (12, 26, 9) 或 (6, 13, 5) | 當沖/短線 |
| 月線 | (12, 26, 9) | 超長線多空判斷 |

---

## 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 只看金叉死叉 | 還要看零軸位置、柱狀圖變化、背離 |
| 在盤整中用 MACD | 盤整時 MACD 容易反覆交叉，應暫停使用 |
| 忽略背離訊號 | 背離是 MACD 最強的訊號 |
| 不區分零軸上下 | 零軸上方的金叉可靠度遠高於零軸下方 |
