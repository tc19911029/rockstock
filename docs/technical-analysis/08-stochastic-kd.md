# KD 隨機震盪指標 (Stochastic Oscillator)

> 由 George Lane 於 1950 年代發明。
> 靈敏度極高的動量指標，特別適合短線交易尋找進出場時機。

---

## 計算方式

### Step 1：計算 RSV (Raw Stochastic Value)
```
RSV = (今日收盤價 - 最近 N 日最低價) / (最近 N 日最高價 - 最近 N 日最低價) × 100
```

### Step 2：計算 %K 和 %D
```
%K = 前日 %K × 2/3 + 今日 RSV × 1/3    （%K 是 RSV 的平滑值）
%D = 前日 %D × 2/3 + 今日 %K × 1/3     （%D 是 %K 的平滑值）
```

### 標準參數
- RSV 週期：9 日
- %K 平滑：3 日
- %D 平滑：3 日
- 記作 KD(9, 3, 3)

### 白話文
- KD 值衡量的是：目前的收盤價在最近 N 日的高低價區間中的「相對位置」
- 值越高 → 收盤價越接近最近的最高價 → 越「強勢」
- 值越低 → 收盤價越接近最近的最低價 → 越「弱勢」

---

## 超買/超賣區間

| KD 範圍 | 狀態 | 意義 |
|:---:|:---:|------|
| > 80 | **超買區** | 短期漲幅過大，注意回調 |
| 20 ~ 80 | **中性區** | 正常波動 |
| < 20 | **超賣區** | 短期跌幅過大，注意反彈 |

### 極端區域
- KD > 90：極度超買，反轉機率很高
- KD < 10：極度超賣，反彈機率很高

## 交叉訊號

### K 線上穿 D 線（金叉）— 買入訊號
```
%K 由下往上穿越 %D

訊號強度：
  ★★★ 在超賣區（<20）的金叉 → 最強買入訊號
  ★★  在中性區的金叉 → 一般訊號
  ★   在超買區（>80）的金叉 → 弱訊號，不推薦
```

### K 線下穿 D 線（死叉）— 賣出訊號
```
%K 由上往下穿越 %D

訊號強度：
  ★★★ 在超買區（>80）的死叉 → 最強賣出訊號
  ★★  在中性區的死叉 → 一般訊號
  ★   在超賣區（<20）的死叉 → 弱訊號，不推薦
```

## KD 背離

### 頂背離（看跌）
- 價格創新高，但 KD 沒有創新高
- 意義：上漲動能衰竭
- 尤其在 KD 超買區出現時特別可靠

### 底背離（看漲）
- 價格創新低，但 KD 沒有創新低
- 意義：下跌動能衰竭
- 尤其在 KD 超賣區出現時特別可靠

## KD 鈍化現象

### 什麼是鈍化？
在強趨勢中，KD 會長期停留在超買或超賣區，失去指示功能。

```
強多頭：
KD ──── 85 ── 90 ── 88 ── 92 ── 87 ── 91 ────
                                                   KD 一直在 80 以上
→ 這不代表要賣出！是「高檔鈍化」= 趨勢非常強

強空頭：
KD ──── 15 ── 10 ── 18 ── 12 ── 8 ── 15 ────
                                                   KD 一直在 20 以下
→ 這不代表要買入！是「低檔鈍化」= 趨勢非常弱
```

### 如何應對鈍化？
1. **先看趨勢：** 在明確趨勢中，不要單靠 KD 逆勢操作
2. **等待訊號：** 等 KD 離開超買/超賣區後的交叉才是真正的訊號
3. **搭配 MACD：** 用 MACD 判斷大趨勢，KD 只用來找進場時機

## KD vs RSI 比較

| 比較項目 | KD | RSI |
|----------|-----|-----|
| **靈敏度** | 高（更容易到極端值） | 中（較平滑） |
| **計算基礎** | 收盤價在高低區間的位置 | 上漲/下跌幅度的比率 |
| **最佳場景** | 短線交易、盤整行情 | 中線波段、趨勢行情 |
| **假訊號** | 較多（因為靈敏） | 較少 |
| **鈍化問題** | 嚴重（常見高/低檔鈍化） | 較輕 |
| **超買/超賣** | 80/20 | 70/30 |
| **訊號速度** | 較快 | 較慢 |

### 選哪個？
- **短線操作** → 用 KD（反應快）
- **波段操作** → 用 RSI（穩定）
- **最佳做法** → 兩個都看，互相確認

## 實戰交易規則

### 盤整行情策略
```
買入：KD < 20 + K 上穿 D（超賣區金叉）
賣出：KD > 80 + K 下穿 D（超買區死叉）

適用：橫盤震盪行情
不適用：明確趨勢行情（會被鈍化騙）
```

### 結合趨勢策略
```
多頭趨勢中（MA 多頭排列）：
  只做多：KD 回到超賣區後金叉 → 買入
  不做空：超買區死叉只是獲利了結，不反手做空

空頭趨勢中（MA 空頭排列）：
  只做空：KD 回到超買區後死叉 → 賣出
  不做多：超賣區金叉只是空頭反彈，不追
```

## 週期參數調整

| 場景 | RSV 週期 | 特點 |
|------|:---:|------|
| 極短線 | 5 | 非常敏感，訊號多但假訊號也多 |
| 短線（標準） | 9 | 最常用的設定 |
| 波段 | 14 | 較平滑，減少假訊號 |
| 中線 | 21 | 訊號少但可靠度高 |

---

## 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| KD 一超買就賣 | 強勢中 KD 會鈍化，要先看趨勢 |
| KD 一超賣就買 | 弱勢中 KD 會鈍化，要先看趨勢 |
| 只看 KD 交叉 | 要看交叉發生在哪個區域（超買/超賣/中性） |
| 在趨勢行情中用 KD | KD 最適合盤整行情，趨勢行情用 MACD 更好 |
